מה זה Processing?

עד עכשיו הארדואינו הדליק נורות והשמיע צלילים. עכשיו נלמד כלי אחר וקסום: Processing — שפה שבה כותבים קוד שמצייר על מסך המחשב. אפשר לצייר צורות, ליצור אנימציות, ואפילו לבנות משחקים — ולחבר הכול לחיישנים של הארדואינו! 🎮

1 קוד שמצייר במקום להדליק נורות

Processing הוא כמו דף ציור שהמחשב מצייר עליו לפי ההוראות שלנו. במקום pinMode ו־digitalWrite, יש פקודות ציור: ellipse (עיגול), rect (מלבן), line (קו), fill (צבע).

🖍️

ההפתעה הנחמדה: Processing דומה מאוד לארדואינו! גם כאן יש setup() שרץ פעם אחת, ו־draw() שרץ שוב ושוב (בדיוק כמו loop). מי שלמד ארדואינו — כבר חצי יודע Processing.

2 הציור הראשון

הנה תוכנית Processing שמציירת עיגול שרודף אחרי העכבר. נסו — זה ממש קצר:

FirstSketch.pde
void setup() {
  size(400, 400);   // גודל חלון הציור
}

void draw() {
  background(20);          // רקע כהה (נמחק כל פעם)
  fill(120, 100, 255);  // צבע סגול (אדום,ירוק,כחול)
  ellipse(mouseX, mouseY, 60, 60);  // עיגול איפה שהעכבר
}
🖱️

mouseX ו־mouseY הם המקום של העכבר. כי draw() רץ שוב ושוב, העיגול "רודף" אחרי העכבר בזמן אמת!

3 לחבר את Processing לארדואינו

כאן מגיע הקסם האמיתי: הארדואינו שולח נתונים מחיישן (למשל פוטנציומטר) דרך ה־Serial, ו־Processing מקבל אותם ומצייר לפיהם. ככה חיישן אמיתי שולט בציור על המסך!

📤 צד הארדואינו

שולח מספר דרך Serial.println() — בדיוק כמו שכבר עשינו בפרק "מדברים עם המחשב".

📥 צד Processing

פותח את אותה יציאה (Serial), קורא את המספר, ומצייר לפיו — עמודה, קו, או תזוזה.

⚠️

חשוב: רק תוכנית אחת יכולה להשתמש ביציאה בכל רגע. אם ה־Serial Monitor של ארדואינו פתוח — סגרו אותו לפני שמריצים את Processing.

4 איך מתחילים — צעד אחר צעד

  1. מורידים את Processing בחינם מהאתר processing.org ומתקינים (זו תוכנה נפרדת מהארדואינו).
  2. מעלים לארדואינו את קוד השליחה (למשל שליחת ערך פוטנציומטר ב־Serial).
  3. סוגרים את ה־Serial Monitor של הארדואינו כדי לשחרר את היציאה.
  4. פותחים ב-Processing את דוגמת ה־.pde מהערכה, ומעדכנים את שם היציאה (COM) לזו של הלוח.
  5. לוחצים Run (▶) — וחלון הציור נפתח וזז לפי החיישן. וואו! 🎉

מוכן ליצור? 🎨

Processing פותח דלת לעולם של אמנות דיגיטלית ומשחקים — והכול מתחבר לחיישנים שכבר הכרת. חזור למדריך הראשי כדי לבנות עוד, או הורד את Processing והתחל לצייר!
