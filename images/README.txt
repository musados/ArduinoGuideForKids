תיקיית תמונות למדריך  /  Images folder for the guide
=====================================================

המדריך (Madrich_Arduino_Ivrit.html) יכול להציג בכל פרויקט,
בתוך הקטע "תרשים חיבורים", שתי תמונות נוספות בלשוניות (טאבים):
  • "התרשים המקורי"  — התמונה מהמדריך המקורי של Freenove
  • "הרכיבים"        — תמונות החלקים של הפרויקט

כדי שהתמונות יופיעו, שמרו כאן (בתיקייה images) קבצים בשמות הבאים.
פשוט חתכו/צלמו את התמונה מה-Tutorial.pdf שקיבלתם עם הערכה, לשימוש אישי,
ושמרו בשם המתאים. הקובץ יכול להיות .png או .jpg.

מוסכמת השמות  /  Naming convention
-----------------------------------
  התרשים המקורי:  <מזהה הפרויקט>_schema.png   (או .jpg)
  תמונת רכיבים:   <מזהה הפרויקט>_parts.png    (או .jpg)

הערות  /  Notes
---------------
• אם קובץ חסר, המדריך פשוט יראה הודעה ידידותית עם שם הקובץ הנדרש.
• פרויקטי הבונוס (b1..b10) הם המצאה שלנו — אין להם "תרשים מקורי",
  אבל אפשר להוסיף להם תמונת רכיבים בשם b1_parts.png וכו'.
• התמונות של Freenove מוגנות בזכויות יוצרים; השתמשו בהן לשימוש אישי בלבד.
  קובץ PartsList.jpg עם כל החלקים נמצא בתיקיית הערכה של Freenove.
